# Curio — Product Overview

**Curio** is an AI-powered personal shopping assistant that helps users discover fashion they'll love through natural conversation and visual search. Instead of scrolling through hundreds of listings, users simply talk to Curio — and it finds the right products, learns their taste, and gets smarter with every message.

---

## Target Users

- Fashion-conscious shoppers who want curation, not just search
- Busy buyers who want outfit recommendations without browsing fatigue
- Style explorers who want to describe a vibe and get exact matches

---

## Features

### Feature 1 — Conversational Shopping

Users describe what they're looking for in plain language — "something casual for brunch but a little elevated" — and Curio responds like a knowledgeable personal stylist. No filters, no dropdowns. Just a conversation.

### Feature 2 — Real-Time Streaming Responses

Curio's replies appear word-by-word as the AI generates them, giving users instant feedback rather than a loading spinner. The experience feels live, not like waiting for a server response.

### Feature 3 — Inline Product Recommendations

When Curio mentions specific items, product cards appear directly inside the chat — with images, prices, ratings, and size selectors. Users never have to leave the conversation to browse.

### Feature 4 — Visual Style Search

Users can upload any fashion photo — a screenshot, a runway look, a street style shot — and Curio extracts the style, color palette, and silhouette using multimodal AI (Gemma4:26b). It then surfaces matching products from the catalog.

### Feature 5 — Live Style Profile

As the conversation progresses, Curio quietly extracts preferences — preferred colors, styles, occasions, and budget — and displays them in a real-time Style Profile panel. Users can see exactly what the AI has learned about their taste.

### Feature 6 — Budget-Aware Recommendations

When a user mentions their budget ("under $150"), Curio automatically filters recommendations to match. Products that exceed the stated budget are deprioritized in the ranking.

### Feature 7 — Multi-Turn Conversation Memory

Curio remembers the full conversation within a session. If a user says "actually, I prefer navy" five messages later, the AI recalibrates all subsequent recommendations — no need to repeat context.

### Feature 8 — Horizontal Product Carousel

Product recommendations are displayed in a horizontally scrollable carousel embedded in the chat. Users can scroll through options without disrupting the conversation flow.

### Feature 9 — Photo Upload with Preview

Users can upload a photo directly in the chat input bar. A thumbnail preview appears before submission, and a loading state is shown while the AI analyzes the image.

### Feature 10 — Mobile-First Responsive Design

On desktop, the Style Profile panel is fixed in a sidebar. On mobile, it collapses into a slide-up drawer accessible via a button — keeping the chat experience full-screen on small devices.

---

## Value Proposition

| Traditional Search | Curio AI |
|---|---|
| User knows what to search | User describes what they want |
| Filter by category/price | AI learns preferences naturally |
| Static product grid | Products appear inside the conversation |
| No memory between sessions | Style profile builds over time |
| Text search only | Text + image search |
